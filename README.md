# LanguageCreator

Bu depo, "Gelişmiş Dil Yaratıcı" projesine aittir. Depoda proje arayüzü için bir HTML dosyası ve geliştirme notlarını içeren `docs/` klasörü bulunur.

- `index (3).html`: Web uygulamasının ana arayüzü.
- `docs/development_research_tr.md`: Araştırma notlarının kısa özeti.
- `docs/development_research_full_tr.md`: Kullanıcının sağladığı ayrıntılı geliştirme alanları araştırması.
